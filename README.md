# ALGORITHMS-LAB-V-SEM

This repository contains the problems discussed in Algorithms Lab III-Year I-Sem at Vignan's Institute of Information Technology Visakhapatnam.

## LIST OF EXPERIMENTS

| Ex  | Name of the Experiment                                             | Topics Covered                |
| --- | ------------------------------------------------------------------ | ----------------------------- |
| 1   | Implement recursive binary search algorithm                        | Divide and Conquer technique  |
| 2   | Implement recursive quick sort algorithm                           | Divide and Conquer technique  |
| 3   | Implement recursive merge sort algorithm                           | Divide and Conquer technique  |
| 4   | Implement randomized quick sort algorithm                          | Divide and Conquer technique  |
| 5   | Find optimal solution for a Knapsack problem                       | Greedy method                 |
| 6   | Find the shortest path using single source shortest path algorithm | Greedy method                 |
| 7   | Implement Huffman coding technique                                 | Greedy method                 |
| 8   | Implement 0/1 Knapsack problem                                     | Dynamic programming technique |
| 9   | Find the shortest path using all pairs shortest path algorithm     | Dynamic programming technique |
| 10  | Implement traveling salesperson problem                            | Dynamic programming technique |
| 11  | Implement sum of subsets                                           | Backtracking technique        |
| 12  | Implement N-Queen’s problem                                        | Backtracking technique        |
